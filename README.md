# sweetapi-php-client
Unofficial client library

## install as composer dependency
```bash
{
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/MetuMaribe/sweetapi-php-client"
        }
    ],
    "require": {
        "MetuMaribe/sweetapi-php-client": "dev-master"
    }
}
```
